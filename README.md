# Task Tracker Terminal
 A task tracker app / table made for the terminal using Python
 
 # Libraries used-
sqlite3 
typing 
rich
datetime

## Instructions-

`git clone`

`cd task_tracker`

### Displaying the table-
`python main.py display`
 
### Adding a task-
`python main.py add "task" "category"`
